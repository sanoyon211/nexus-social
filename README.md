## 📖 Project Overview

**The Nexus Social** is a social media web application that allows users to connect with others, share posts, and interact through a clean and responsive interface. Built with React for dynamic UI rendering and styled with Tailwind CSS for a modern look and feel.

---

## 🛠️ Tech Stack

| Technology | Purpose |
|---|---|
| React.js | Component-Based UI & State Management |
| JavaScript (ES6+) | Logic & Interactivity |
| Tailwind CSS | Utility-First Responsive Styling |
| HTML5 | Markup Structure |
| CSS3 | Custom Styles |

---

## ✨ Key Features

- 👤 User profile and feed UI
- 📱 Fully **responsive** design for all screen sizes
- ⚡ Fast SPA experience with React Router
- 🎨 Clean and modern social media UI/UX
- 🔄 Dynamic content rendering with React components

## 🚀 Run Locally

```bash
# 1. Clone the repository
git clone https://github.com/sanoyon211/nexus-social.git

# 2. Navigate into the project folder
cd nexus-social

# 3. Install dependencies
npm install

# 4. Start the development server
npm run dev
# OR
npm start

# 5. Open in browser
# Visit: http://localhost:5173 or http://localhost:3000
```
